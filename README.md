ğŸ® How to Play

1. Run the program using the command above
2. The computer will secretly choose a random number between 1 and 100
3. Enter your guess when prompted
4. The game will give you hints:
   - "Higher number please" - if your guess is too low
   - "Lower number please" - if your guess is too high
5. Keep guessing until you find the correct number!
6. The game will show you how many attempts it took

Example Gameplay:
```
Guess the number: 50
Higher number please
Guess the number: 75
Lower number please
Guess the number: 63
You guess the number 63 correctly in 3 attempts
```
